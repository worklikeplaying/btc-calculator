이용방법

비트코인 계산기의 모든 기능을 단계별로 알아보세요.

기본 사용법

  • 1

    실시간 가격 확인

    메인 화면에서 업비트(KRW)와 바이낸스(USD) 기준 비트코인 가격을 1초마다 자동으로 갱신해 확인할 수 있습니다. 상단의 KRW / USD 토글로 주 표시 화폐를 전환하세요.

  • 2

    김치 프리미엄 파악

    가격 카드 아래에 김치 프리미엄(%)이 실시간으로 표시됩니다. 양수(빨간색)는 국내 거래소가 해외보다 비싼 상태, 음수(파란색)는 저렴한 상태를 의미합니다.

  • 3

    BTC 환산 계산기 사용

    BTC 수량을 입력하면 즉시 KRW·USD 환산 금액이 표시됩니다. 버튼 하나로 사토시(Satoshi) 단위로 전환해 소액 거래도 계산할 수 있습니다.

  • 4

    프리미엄 반영 계산

    계산기 하단의 "프리미엄 반영 계산" 영역에서 원하는 프리미엄(%)을 직접 입력하거나, "김프 적용" 버튼을 눌러 현재 김치 프리미엄을 자동으로 불러올 수 있습니다.

  • 5

    결과 복사

    각 계산 결과 옆의 "복사" 버튼을 누르면 숫자 값이 클립보드에 저장됩니다. 쉼표·통화 기호 없이 순수 숫자로 복사됩니다.

김치 프리미엄이란?

김치 프리미엄(Kimchi Premium)은 국내 암호화폐 거래소의 비트코인 가격이 해외 거래소보다 높을 때 발생하는 가격 차이를 말합니다.

계산 공식:

((국내 KRW 가격 ÷ (해외 USD 가격 × USD/KRW 환율)) − 1) × 100 = 김치 프리미엄(%)

예를 들어 결과가 +3.5%라면, 국내에서 비트코인이 해외보다 3.5% 더 비싸게 거래되고 있음을 의미합니다.

사토시(Satoshi) 단위

사토시는 비트코인의 가장 작은 단위로, 1 BTC = 100,000,000 사토시(1억 사토시)입니다.

소액 결제나 수수료 계산 시 사토시 단위가 더 편리합니다. 계산기에서 "사토시" 버튼을 눌러 단위를 전환하세요.

데이터 출처

  • BTC/KRW 가격: 업비트(Upbit) 공개 API
  • BTC/USD 가격: 바이낸스(Binance) 공개 API
  • USD/KRW 환율: ExchangeRate-API 공개 API

모든 데이터는 1초마다 자동 갱신되며, API 오류 발생 시 화면 상단에 안내 메시지가 표시됩니다.
