개인정보처리방침

최종 업데이트: 2026년 5월 1일

1. 총칙

비트코인 계산기(이하 "서비스")는 사용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다. 본 방침은 서비스 이용 과정에서 수집되는 개인정보의 처리 방법을 설명합니다.

2. 수집하는 개인정보

본 서비스는 기본적으로 개인정보를 수집하지 않습니다. 서비스는 외부 공개 API를 사용하여 암호화폐 가격 데이터를 표시하며, 이 과정에서 사용자 개인정보는 수집되지 않습니다.

문의 접수 시 수집 항목:

  • 이메일 주소
  • 문의 내용 (사용자가 직접 입력한 정보)

3. 개인정보의 이용 목적

수집한 개인정보는 다음 목적에 한해 사용됩니다.

  • 사용자 문의에 대한 답변 및 처리
  • 서비스 개선을 위한 피드백 분석

4. 개인정보 보유 기간

문의를 통해 수집된 개인정보는 문의 처리 완료 후 6개월 이내에 삭제됩니다. 단, 관련 법령에 의한 보존 의무가 있는 경우 해당 기간 동안 보관합니다.

5. 쿠키 및 로컬 스토리지

본 서비스는 테마 설정(다크/라이트 모드)을 저장하기 위해 브라우저 로컬 스토리지를 사용합니다. 이 데이터는 사용자의 기기에만 저장되며, 서버로 전송되지 않습니다.

6. 제3자 API 사용

본 서비스는 가격 데이터 수집을 위해 다음 외부 API를 사용합니다.

  • 업비트 공개 API (api.upbit.com)
  • 바이낸스 공개 API (api.binance.com)
  • ExchangeRate-API (open.er-api.com)

각 서비스의 개인정보처리방침은 해당 사이트에서 확인하시기 바랍니다.

7. 사용자의 권리

사용자는 언제든지 자신의 개인정보에 대해 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 수정 요청
  • 개인정보 삭제 요청
  • 개인정보 처리 정지 요청

권리 행사는 문의 페이지를 통해 요청하실 수 있습니다.

8. 방침 변경

본 개인정보처리방침이 변경될 경우, 서비스 내 공지를 통해 안내드립니다. 변경된 방침은 공지된 날로부터 효력이 발생합니다.
